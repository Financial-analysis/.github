## 미래의 내가 보내는 소비 피드백 – AI 기반 금융 넛지 서비스

<div align="center">
  <h1>SKHU LINK - 더 나은 성공회대생의 수업 경험을 위한 강의 커뮤니티 서비스</h1>
  <p>🔍 서비스 설명 🔍</p>
  
  이 서비스는 사용자의 **현재 소비 데이터를 분석해, 동일한 소비 습관이 지속될 경우 도달하게 될 미래의 재정 상태를 미리 보여주는** AI 금융 넛지 어플리케이션입니다.
숫자 중심의 가계부가 아닌, **'미래의 나'라는 관점에서 소비 결과를 해석하고 피드백을 전달**함으로써 사용자는 스스로 소비를 돌아보고 행동을 조정하도록 돕는 것을 목표로 합니다.
